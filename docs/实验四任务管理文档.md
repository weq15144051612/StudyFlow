## 6. 接口变更记录

### 6.1 Task 实体新增方法

| 方法名 | 签名 | 功能说明 |
|-------|------|---------|
| `addTag` | `addTag(tag: string): void` | 为任务添加标签（自动去重） |
| `removeTag` | `removeTag(tag: string): void` | 从任务中移除标签 |
| `hasTag` | `hasTag(tag: string): boolean` | 检查任务是否包含指定标签 |
| `getTags` | `getTags(): string[]` | 获取任务的所有标签（返回副本） |

### 6.2 TaskRepository 新增方法

| 方法名 | 签名 | 功能说明 |
|-------|------|---------|
| `findByTag` | `findByTag(userId: string, tag: string): Task[]` | 按单个标签查询任务 |
| `findByTags` | `findByTags(userId: string, tags: string[], matchAll: boolean): Task[]` | 按多个标签查询（matchAll=true 表示全部匹配） |

### 6.3 TaskService 新增方法

| 方法名 | 签名 | 功能说明 |
|-------|------|---------|
| `addTag` | `addTag(taskId: string, tag: string): Task` | 为任务添加标签 |
| `removeTag` | `removeTag(taskId: string, tag: string): Task` | 移除任务的指定标签 |
| `listTasksByTag` | `listTasksByTag(userId: string, tag: string): Task[]` | 按标签筛选任务列表 |
